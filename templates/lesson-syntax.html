When using the syntax, remember not to include the curly braces ({}) in your code.

Text

The Text widget creates a standard paragraph text.

For the text widget, the only thing that is needed is the text. Create a text widget using the Create slider and Create button, and just input the text into the entry that appears.

Title

The Title widget creates a strong text.

For the title widget, the only thing that is needed is the text. Create a title widget using the Create slider and Create button, and just input the text into the entry that appears.

Pause

The Pause widget prompts the user to click 'Continue' before continuing the text.

For the pause widget, no code is needed. Just make the widget and leave it.

Hint

The Hint widget creates a button that, when clicked, reveals a hint.

Simply input the hint text into the entry that appears after creating the widget.

Slider

The Slider widget lets the user change the value of a variable using a slider to see how it affects an equation.

The slider widget code takes in a variable name and a minimum and maximum value.

{varname(oneletter)}{minimumValue},{maximumValue} Example: x1,25

Equation

The Equation widgets lets the user see how different values of variables affect an equation.

The equation widget code takes in just one parameter: the equation, with all operators and variables separated by spaces.

Example: x * 2 Note on the example: For the above example to work, you must have either a NumberEntry widget or a Slider widget with the variable name of x.

NumberEntry

The NumberEntry widget lets the user change the value of a variable using an Entry to see how it affects an equation.

The NumberEntry takes in the same parameters as the Slider (see above). The example for the slider also works for the NumberEntry.

RadioQuiz

The RadioQuiz widget prompts the user to respond to a question by checking one radio button.

The code for a RadioQuiz takes in the correct answer's index, the question, and all the options. _{correctIndex}|{question}|{option0}|{option1} You can have as many options as you want, just separate them by bars (|).

CheckboxQuiz

The CheckboxQuiz widget prompts the user to respond to a question by checking any number of boxes. The quiz will say that the user is correct if they select all the correct indexes.

The code for a CheckboxQuiz takes in the correct answers' indexes, the question, and all the options. _{correctIndex1}+{correctingIndex2}|{question}|{option0}|{option1} You can have as many options as you want, just separate them by bars (|). You can also have as many correct indexes as you want, just separate them by plus signs (+).

EntryQuiz

The EntryQuiz widget prompts the user to respond to a question by entering in a response.

The code for an EntryQuiz takes in just the correct answer and the question. _{answer}|{question}
